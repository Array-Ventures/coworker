# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

All participants are expected to uphold this code. Please report unacceptable behavior via GitHub's private reporting feature or by opening a confidential issue.

The full text is available at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
